一个小工具用于监测静态网页

同学今年考研，在等待中传的录取结果，帮助写了一个小工具监测中传研招网的更新内容

思路

第一种：

  1. 可以向服务器发送一个 If-Modified-Since的请求（ http://www.w3.org/Protocols/rfc2616/rfc2616-sec14.html ）。
  2. 把 a.html 的修改时间写到请求里。
  3. 如果网页更新了，服务器会返回一个 200 的应答，这时就重新抓取网页，更新本地文件。
  4. 如果网页没有更新，服务器会返回一个304的应答。这时就不需要更新文件了。

第二种：
就是爬虫，爬取页面内容或标题，来判断是否有更新

代码

这里采用爬虫的思路
利用python的urlib，chardet获取网页页面，然后用BeautifulSoup来解析文本内容，最后利用stmp协议，当发现网页有更新后发送邮件给指定邮箱。
